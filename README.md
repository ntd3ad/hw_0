# hw_0
## Распаковка строки

Создать Go функцию, осуществляющую примитивную распаковку строки, содержащую повторяющиеся символы / руны, например:

1.  `a4bc2d5e`    => `aaaabccddddde`
2.  `a12`         => `aaaaaaaaaaaa`
3.  `abcd`        => `abcd`
4.  `45`          => `` (некорректная строка)
5.  `qwe\4\5`     => `qwe45` (*)
6.  `qwe\45`      => `qwe44444` (*)
7.  `qwe\\5`      => `qwe\\\\\` (*)
8.  `qw0e`        => `qe`
9.  `<5`          => `<<<<<`
10. `☀️5`          => `☀️☀️☀️☀️☀️`
